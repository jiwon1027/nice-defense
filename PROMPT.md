# NICE 통합서버 디펜스 — 제작 프롬프트

> 사내 AI 경진대회 출품용 웹 미니게임 제작 프롬프트.
> 단일 HTML 파일(Canvas + Vanilla JS)로 한 번에 생성 요청용.

---

## 한 줄 요약
'뱀파이어 서바이버즈 / 탕탕특공대' 스타일의 **레벨업 증강 선택** 디펜스 게임을, NICE정보통신의 결제·VAN·PG 업무 용어로 테마화하여 단일 HTML 파일로 제작한다.

---

## 게임 기본 콘셉트 (뱀파이어 서바이버즈식 이동/회피)
1. 플레이어는 **NICE 기동형 보안 유닛**(파란 사각형)으로, **WASD/방향키**로 자유 이동한다(마우스 홀드 이동도 지원). 이동 속도는 빠르게.
2. 화면 사방에서 **결제 데이터**(빨강/노랑/초록 블록)가 플레이어를 추격한다. 일부(이상거래·보스)는 **총알을 발사** → 플레이어는 이동하며 **총알을 피해야** 한다.
3. 자동 무기(레이저/QR/폭탄/미사일)와 **플레이어 주위를 도는 궤도 방화벽 쉴드**가 데이터를 **처리(승인)** 한다.
4. 데이터 처리 시 **매출(점수)** 과 **경험치(XP)** 가 오른다. XP 바는 화면 상단에 표시.
5. 적·총알에 맞으면 **서버 HP**가 깎이고 '피격!' 텍스트가 뜬다(짧은 무적 i-frame 부여). HP 0이면 게임 오버.

## 핵심: 레벨업 증강 시스템
1. XP 바가 100%가 되면 게임이 **일시정지**되고 화면 중앙에 **'NICE 기술 증강 선택'** 팝업이 뜬다.
2. 팝업에 무작위 **3장 카드**가 나타나고, 클릭하면 능력치가 즉시 반영된 후 게임 재개.
3. 레벨업마다 데이터가 몰려오는 **속도와 양**이 조금씩 빨라진다.
4. 증강은 **등급(rarity) 가중추첨**: 일반 가맹점 60% / 우수 가맹점 30% / 프리미엄 가맹점 10%.

## 무기 8종 (데이터 배열로 모듈화, Lv 1~5 강화)
| 무기 | 메커니즘 |
|---|---|
| 🛡 방화벽 쉴드 | 플레이어를 도는 육각 궤도 실드(기본, 접촉 처리+넉백, Lv↑마다 개수↑) |
| ⚡ 승인 레이저(VAN) | 최근접 적 자동 조준 연사(빔 트레이서+머즐플래시) |
| 📡 QR 스캐너 | 원형 광역(AoE) 스캔 펄스(2중 링+눈금, 적 둔화) |
| 💣 정산 배치 폭탄 | 주기적 폭탄 투하 → 강력한 범위 폭발(화면 흔들림) |
| 🎯 FDS 추적 미사일 | 적 자동 추적 유도탄(꼬리 트레일) |
| ⛓ TRS 거래중계 체인 | 거래가 인접 노드로 연쇄 라우팅되는 체인 라이트닝(TRS) |
| 🛰 펌뱅킹 드론 | 주위를 도는 자동 포탑 드론(최대 3기) |
| 📲 NFC 태그 필드 | 플레이어 주위 접촉 구역 지속 광역 처리 |

## 증강 (20종 이상, 패시브/특수/무기해금 혼합)
초고속 VAN 서버(공속+) · VAN 직승인 회선(피해+) · 광대역 네트워크(범위+) · PG 이중화 방화벽(최대HP+회복) · IDC 이중화 상일·오쇠(최대HP+40%+재생) · 5G 엣지 전용망(이동속도+) · 토큰화 보안(관통+) · 간편결제 멀티채널(투사체+) · 캐시백 리워드(처리 시 회복) · 매입 정산 최적화(매출×) · 정산 배치 처리(XP+) · UPS 무정전 전원(HP 재생) · 부정거래 차단 FDS룰(크리티컬) · 트래픽 셰이핑(적 둔화 오라) · 망분리 배리어(주기적 무적) · 무중단 서비스 SLA(1회 부활) · 무기 7종 해금/강화 카드.

> NICE 실제 용어 적극 차용: **VAN / PG / IDC(상일·오쇠) / TRS / FDS / 펌뱅킹 / 토큰화 / 정산·매입**.
> 카드마다 `💡 업무 ·` 한 줄로 실제 업무 용어를 설명 → 유용성(교육효과) 점수 근거.

## 맵 아이템 (적·보스 처치 시 드랍, 밟으면 획득)
| 아이템 | 효과 |
|---|---|
| 🧪 백업 복구 팩 | 최대 HP의 30% 즉시 회복 |
| 🧰 정산 금고 | 매출 보너스 +₩800 & 즉시 레벨업 1회(증강 선택) |
| 🧹 일괄 정산 | 화면의 모든 데이터를 일괄승인(전체 몹 제거) + 화면 플래시 |

## 이름 입력 & 명예의 전당(스코어보드)
- 시작 화면에서 **닉네임 입력**(최대 12자, Enter로 시작). 시작/게임오버 화면에 **🏆 명예의 전당 TOP 10** 표시.
- **localStorage**로 저장 → 새로고침/재실행해도 기록 유지. **매출(점수) 내림차순** 정렬, 본인 기록 하이라이트 + 순위 표시.
- ⚠️ 한계: localStorage는 **같은 PC·브라우저 내에서만** 공유(서버 없음). **행사장 PC 1대(키오스크)** 운영에 최적. 여러 대 합산이 필요하면 별도 서버/DB 필요.
- 이름은 HTML 이스케이프 처리(XSS 방지).
- **다음 참가자 플로우**: 게임오버 시 버튼은 **"다음 참가자 ▶"** — 누르면 곧바로 재시작이 아니라 **시작 화면(이름 입력)으로 복귀** + 이름칸 비우고 포커스. 키오스크에서 다음 사람이 새 이름으로 바로 시작.

## 시스템 고도화
- 🔥 일정 주기 **블랙프라이데이 트래픽 폭주** 웨이브 + **대규모 망장애 보스** 등장(보스 처치 시 아이템 확정 드랍)
- 🔄 **재승인 요청(리롤)** 1회 — 카드 다시 뽑기
- 플로팅 텍스트: "승인 ₩100" / "피격! -10" / "캐시백 +2HP" / "CRIT!"
- 난이도 튜닝: 적 탄막 **발사 빈도 낮춤**(회피 난이도↓), 대신 **적 이동속도 소폭↑**

## 디자인 요구사항
- 배경: 깔끔한 **다크 그레이** + 미세 그리드
- 포인트 컬러: NICE 메인 블루 **`#0046ff`** 계열
- 폰트: 고딕 계열(Pretendard/Apple SD Gothic Neo), 게임 화면 **풀스크린**
- 게임 오버 시 **최종 매출 / 도달 레벨 / 생존 시간**을 표시하는 결과창

## 기술 제약
- **단일 HTML 파일** (Canvas 2D + Vanilla JS, 외부 의존성 0)
- 더블클릭 실행 / 인터넷·빌드 불필요 (사내 PC 시연 안전)
